# CodeReview Agent — 智能代码安全审查助手

基于 Coze 平台搭建的智能代码审查 Bot，自动检测代码中的 SQL 注入、空指针异常、硬编码密钥等常见安全问题，按高危 / 中危 / 建议三级输出审查报告。

## 核心功能

- **双层审查架构**：规则引擎（正则匹配）精确命中可模式化问题 + LLM 语义分析兜底处理逻辑缺陷
- **三级风险报告**：高危（必须修复）/ 中危（建议修复）/ 建议（代码规范）
- **即时反馈与修复建议**：用户粘贴代码，Bot 直接输出结构化审查报告，附带可操作的修复方案

## 技术架构

```
用户输入代码 → Coze 工作流 → LLM 语义分析层
                              → 规则引擎层
                              → 合并输出三级审查报告
```

## 检测能力

| 风险类型           | 级别   | 检测方式            |
|-------------------|--------|-------------------|
| SQL 注入           | 高危   | 规则引擎 + LLM     |
| 硬编码密码/密钥    | 高危   | 规则引擎           |
| 空指针异常         | 高危   | LLM 语义分析       |
| 未处理的异常       | 中危   | LLM 语义分析       |
| 不安全的文件操作   | 中危   | 规则引擎 + LLM     |
| 硬编码 URL/IP     | 中危   | 规则引擎           |
| 魔法数字           | 建议   | 规则引擎           |
| 命名不规范         | 建议   | LLM 语义分析       |
| 缺少注释和文档     | 建议   | LLM 语义分析       |

## 测试验证

使用 SQL 注入和硬编码密钥两个测试案例验证，均准确命中目标问题并生成可用的修复建议。

## 使用方式

1. 打开 Coze Bot
2. 粘贴你的代码片段
3. 等待 Bot 输出结构化的安全审查报告
4. 根据报告进行修复
